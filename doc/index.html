High Point API

Документация включает себя информацию о HTTP запросах к серверу мобильных клиентов и браузеров, работе с веб-сокетами.

Обозначения

В описании параметров могут использоваться следующие обозначения:

  • ? — параметр может иметь значение null,
  • = — необязательный параметр.

Примеры:

name: string // обязательный строковый параметр
temperature: float // обязательное вещественное число
rating: integer // обязательное целое число
id: uint // обязательное целое положительное число
user: User // обязательный параметр типа User
user: User? // обязательный параметр типа User или null
temperature: float= // необязательное вещественное число
rating: integer? // обязательное целое число или null
name: string?= // необязательный строковый параметр или null
