Текущий пользователь

Получение данных о пользователе

GET
/v201405/me
Требуется авторизация

Ответ:

{
  data: {
    user: MyUser
  }
}

Обновление настроек фильтра текущего пользователя

PUT
/v201405/me/filter
Требуется авторизация
PUT параметры
НазваниеТипОписание
cityIdsstring=Идентификаторы городов, разделенных запятыми
gendersstring=Идентификаторы полов, разделенных запятыми
maxAgeuint=Максимальный возраст
minAgeuint=Минимальный возраст
viewTypeFilterViewType=Режим просмотра пользователей

Ответ:

{
  data: true
}

Добавление должности

POST
/v201405/me/career-posts
POST параметры
НазваниеТипОписание
namestringНазвание должности

Ответ:

{
  data: CareerPost
}

В случае пустого значения параметра name выдается ответ со статусом 403 и данными:

{
  error: {
    code: ErrorCode.WRONG_PARAMS,
    params: [{
      code: ErrorCode.REQUIRED,
      name: 'name'
    }]
  }
}

Удаление должностей

DELETE
/v201405/me/career-posts
POST параметры
НазваниеТипОписание
idsstringИдентификаторы должностей, разделенных запятыми

В ответ выдается список идентификаторов должностей, которые были удалены у пользователя:

{
  data: Array.<uint>
}
