Фотографии

Загрузка фотографии

POST
/v201405/photos/add
Требуется авторизация
POST параметры
НазваниеТипОписание
imageFileФайл изображения

Ответ:

{
  data: {
    photo: Photo
  }
}

В случае неправильного формата файла выводится ошибка со статусом 403:

{
  error: {
    code: ErrorCode.WRONG_FILE_FORMAT
  }
}

В случае слишком большого размера файла выводится ошибка со статусом 403:

{
  error: {
    code: ErrorCode.TOO_LARGE_FILE
  }
}

В случае слишком маленького размера изображения выводится ошибка со статусом 403:

{
  error: {
    code: ErrorCode.TOO_SMALL_SIZE
  }
}

Удаление фотографии

POST
/v201405/photos/<id>/remove
Требуется авторизация
Параметры адреса
НазваниеТипОписание
iduintИдентификатор фотографии

В ответе выдается идентификатор удаленной фотографии:

{
  data: {
    id: uint
  }
}

Возможные ошибки:

  • объект не найден
  • отказано в доступе
